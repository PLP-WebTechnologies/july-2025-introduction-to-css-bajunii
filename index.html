Understanding the building blocks of web styling

Introduction to CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, or in other media.

This page demonstrates various CSS properties and the fundamental concept of the CSS Box Model.

The CSS Box Model

Every element in CSS is essentially a rectangular box. The CSS box model describes how these boxes are structured:

Content Area

  • Content: The actual content of the element
  • Padding: Space between content and border
  • Border: A line around the padding and content
  • Margin: Space outside the border
